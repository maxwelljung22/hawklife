# Contributing to HawkLife

## Contribution Policy

HawkLife is an internal platform built solely for the use of St. Joseph's Preparatory School.

The HawkLife development team is not accepting external contributions to this repository.

This GitHub repository and its documentation exist only for:

- the HawkLife development team
- St. Joseph's Preparatory School administration to observe, review, and audit the platform

External pull requests, forks intended for upstream contribution, or unsolicited code changes are not part of the contribution process for this project.

## Who May Contribute

Direct code changes should only be made by authorized members of the HawkLife development team.

Administrative stakeholders at St. Joseph's Preparatory School may review the repository and documentation for oversight, auditing, and platform understanding, but are not expected to contribute code directly unless specifically participating in the development process.

## Questions, Ideas, or Feedback

If you have ideas, questions, concerns, or would like more information about HawkLife, contact Maxwell Jung '27.

## Repository Intent

This repository is school-specific and is not maintained as an open-source community project.

All branding, workflows, documentation, and implementation details are intended for the internal use, review, and stewardship of HawkLife at St. Joseph's Preparatory School.
